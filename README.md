# Amirai — генератор картинок-подборок аниме

Локальное веб-приложение для создания картинок-подборок аниме под TikTok / Reels / Shorts
и продвижения сайта **amirai.online**. Данные и постеры берутся из **Shikimori API**.
Базы данных нет — всё работает через API в реальном времени.

## Возможности

- **Редактор слайдов**: первый слайд — обложка (название подборки + логотип),
  следующие — сетки с аниме.
- **Поиск аниме** через Shikimori (по русским и английским названиям), постер
  добавляется кликом.
- **Названия редактируются** вручную (по умолчанию подставляются из Shikimori).
- **Перетаскивание карточек** мышкой с **привязкой к сетке** (snap-to-grid),
  изменение размера за угол, кнопка «Разложить по сетке».
- **Два формата**: 9:16 (1080×1920, TikTok) и 1:1 (1080×1080, посты).
- **Экспорт** каждого слайда или всех сразу в PNG.

## Стек

- **Backend**: Python + Flask — отдаёт страницу и проксирует Shikimori
  (нужен корректный `User-Agent`) и постеры (обходит CORS, чтобы canvas
  можно было экспортировать в PNG).
- **Frontend**: HTML + CSS + [Alpine.js](https://alpinejs.dev) +
  [html-to-image](https://github.com/bubkoo/html-to-image) (библиотеки лежат
  локально в `static/vendor`).

## Запуск

### Просто (Windows)

Дважды кликнуть **`start.bat`** — он поставит зависимости, откроет браузер и
запустит сервер.

### Вручную

```powershell
python -m pip install -r requirements.txt
python app.py
```

Открыть <http://127.0.0.1:5000>.

## Как пользоваться

1. Сверху выбери формат: **9:16 TikTok** или **1:1 пост**.
2. **Титульный слайд** (первый): впиши название подборки и подзаголовок,
   при желании включи/выключи логотип и URL, выбери фон.
3. Перейди на **слайд-сетку** (внизу лента слайдов) → найди аниме в поиске →
   кликай по постерам, чтобы добавить карточки.
4. Двигай карточки мышкой (прилипают к сетке), тяни угол для размера,
   меняй число колонок/отступы, жми **«Разложить по сетке»** для ровной раскладки.
5. Правь названия карточек в списке слева.
6. Добавляй слайды кнопками **+ Сетка / + Титул** внизу.
7. **⬇ Слайд** — скачать текущий, **⬇ Скачать всё** — все слайды по очереди.

## Структура

```
app.py                  Flask: страница + /api/search, /api/anime/<id>, /api/image
templates/index.html    Редактор (разметка + Alpine-биндинги)
static/css/style.css    Тёмная тема, красный акцент бренда
static/js/app.js        Логика редактора: слайды, drag/snap, экспорт
static/vendor/          alpine.min.js, html-to-image.js (локально)
static/img/             Логотипы amirai
images/                 Исходные логотипы
```

## Примечания

- Shikimori ограничивает частоту запросов (~5 rps) — в `app.py` есть лёгкий троттлинг.
- Для работы нужен интернет (запросы к Shikimori). Сами библиотеки фронта —
  локальные, CDN не требуется.
- `User-Agent` приложения задаётся в `app.py` (`USER_AGENT`).
