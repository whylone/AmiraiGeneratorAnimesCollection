Вот точные дизайн-токены сайта amirai.online (из tailwind.config.js + main.css). Стиль — брутализм: чёрный фон, белый текст,
  красный акцент, острые углы.

  Цвета (основная палитра)

  ┌────────────────────┬───────────────────────┬─────────────────┐
  │     Назначение     │    HEX / значение     │       Где       │
  ├────────────────────┼───────────────────────┼─────────────────┤
  │ Фон (основной)     │ #000000               │ страница        │
  ├────────────────────┼───────────────────────┼─────────────────┤
  │ Карточки           │ #0a0a0a               │ блоки, постеры  │
  ├────────────────────┼───────────────────────┼─────────────────┤
  │ Вторичный фон      │ #1a1a1a               │ поля, ховеры    │
  ├────────────────────┼───────────────────────┼─────────────────┤
  │ Текст              │ #ffffff               │ основной        │
  ├────────────────────┼───────────────────────┼─────────────────┤
  │ Текст приглушённый │ rgba(255,255,255,0.4) │ подписи         │
  ├────────────────────┼───────────────────────┼─────────────────┤
  │ Акцент (бренд)     │ #ff3333               │ главный красный │
  ├────────────────────┼───────────────────────┼─────────────────┤
  │ Акцент тёмный      │ #cc2929               │ ховер/градиент  │
  ├────────────────────┼───────────────────────┼─────────────────┤
  │ Границы            │ rgba(255,255,255,0.1) │ тонкие рамки    │
  └────────────────────┴───────────────────────┴─────────────────┘

  Акцентный градиент: linear-gradient(135deg, #ff3333, #cc2929)

  Дополнительные (статусы):
  - Успех/зелёный: #22c55e (яркий #66ff66)
  - Светло-красный: #ff6666
  - Telegram-синий: #2AABEE

  Углы (border-radius)

  0px — везде острые углы. Это жёсткое правило сайта (брутализм, никаких закруглений). Для TikTok-шаблонов держи
  прямые углы — это фирменная черта.

  Типографика

  - Шрифт: Inter (фолбэк -apple-system, BlinkMacSystemFont, sans-serif)
  - Веса: 400 / 700 / 900
  - Заголовки: UPPERCASE + разрядка letter-spacing: 0.1–0.15em

  Готовый «root» для твоего шаблонизатора

  :root {
    --bg:        #000000;
    --card:      #0a0a0a;
    --secondary: #1a1a1a;
    --fg:        #ffffff;
    --muted:     rgba(255,255,255,0.4);
    --accent:    #ff3333;
    --accent-dk: #cc2929;
    --border:    rgba(255,255,255,0.1);
    --success:   #22c55e;
    --radius:    0px;
    --font:      'Inter', sans-serif;
    --accent-gradient: linear-gradient(135deg, #ff3333, #cc2929);
  }
